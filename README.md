# Advanced Todo Dashboard

A modern, interactive and fully responsive task management dashboard built with semantic HTML5, advanced CSS3, and Vanilla JavaScript.

This project goes beyond a basic todo application by focusing heavily on:
- advanced UI/UX design
- responsiveness
- accessibility
- component interaction
- polished animations
- professional frontend architecture
